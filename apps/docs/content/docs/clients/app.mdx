## Server URL

The app can use the official Happier server, or you can add one or more custom servers for self-hosting.

### Saved servers + switching

- You can save multiple servers and switch between them.
- Credentials are stored **per server**, so a hosted account and a self-hosted account won’t overwrite each other.
- On different servers, accounts are independent even if you use the same login provider.
- On web, you can switch the server **for this tab** or **make it the device default**.

### Configure-server links (1-click)

Some installers (for example `hstack setup` and Proxmox) print “configure server” links/QRs that open the app/web UI directly on your server configuration screen, prefilled with the correct URL.
