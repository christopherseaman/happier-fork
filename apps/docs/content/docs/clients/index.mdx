---
title: Clients
description: How the CLI and app interact with the server.
---

Happier has two primary clients:

- **CLI** (`@happier-dev/cli`): agent runner + daemon + sync client.
- **App** (`@happier-dev/app`): mobile + web UI for sessions and devices.

If you’re looking specifically for background operation and service installation, see [Daemon](./daemon).

For server selection, profile switching, and multi-server behavior, see [Features → Multi-server](/docs/features/multi-server).
